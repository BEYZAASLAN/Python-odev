 Veri Bilimi <br>
 İlgili veri tabanları CSV formatındadır  <br>
 İlk olarak veri tabanına ait içerikleri incelemeniz  gerekmektedir. <br>

 Projemde Seaborn,MatplotLib,Pandas,Numpy kütüphaneleri kullandım.
